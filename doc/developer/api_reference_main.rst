Developer Reference
===================

Purpose
-------

This is the single landing page for lower-level developer reference.

Use it for two kinds of material:

- curated technical reference organized by topic and internal model grouping;
- autogenerated API documentation organized by package and module.

Curated technical reference
---------------------------

This is the primary reference path. Use it when you need architecture flows,
internal solver groupings, controller implementation-oriented pages, or
entity-level asset reference that is easier to navigate by topic or internal
model grouping than by Python package.

Heat exchanger asset
++++++++++++++++++++

The **Heat Exchanger Asset** is the entity-level representation of an asset
that transfers thermal energy between two hydraulic sides.

It captures the ports and configuration needed to describe a coupled thermal
exchange component in the simulation input model.

Implemented by
--------------

This reference page documents the ``HeatExchanger`` entity in
``omotes_simulator_core.entities.assets.heat_exchanger``.

Ates controller class
=====================================

Purpose
-------

This page documents the controller-side class used to hold control state for
an ATES cluster.

Use this reference for member-level details needed when modifying or extending
controller behavior for aquifer thermal energy storage assets.

For user-facing interpretation of ATES operation, start from :doc:`../controller`
and :doc:`../../physics/ates_cluster_physics`.

Ates asset
+++++++++++++++++++++++++++++++++++++++++++++

The **ATES Asset** is the entity-level representation of an **Aquifer Thermal
Energy Storage (ATES)** system within the simulation input model.

It captures the parameters and ports used to describe seasonal thermal storage
and retrieval before the network and asset behavior are solved.

Implemented by
--------------

This reference page documents the ``AtesCluster`` entity in
``omotes_simulator_core.entities.assets.ates_cluster``.

---------------------

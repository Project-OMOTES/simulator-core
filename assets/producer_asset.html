Producer asset¶

The Producer Asset is the entity-level representation of a controllable heat source in the simulation input model.

It groups the metadata, ports, and operating parameters needed to represent heat production before those values are translated to solver behavior.

Implemented by¶

This reference page documents the ProductionCluster entity in omotes_simulator_core.entities.assets.production_cluster.

Related documentation¶

  • For producer behavior during simulation, see ProductionCluster.

  • For network-level interpretation of pressure and flow, see Network layer.

class omotes_simulator_core.entities.assets.production_cluster.ProductionCluster(asset_name: str, asset_id: str, port_ids: list[str])

A ProductionCluster represents an asset that produces heat.

control_mass_flow: bool

Flag to indicate whether the mass flow rate is controlled. If True, the mass flow rate is controlled. If False, the mass flow rate is not controlled and the pressure is predescribed.

controlled_mass_flow: float | None

The controlled mass flow of the asset [kg/s].

first_time_step: bool

Flag to indicate whether it is the first time step of the simulation.

get_actual_heat_supplied() → float

Get the actual heat supplied by the asset.

Return float:

The actual heat supplied by the asset [W].

heat_demand_set_point: float

The heat demand set point of the asset [W].

is_converged() → bool

Check if the asset has converged with accepted error of 0.1%.

The convergence criteria verifies whether the heat supplied by the asset - based on the solver asset - matches the heat demand set point of the asset.

In other words: |Q_calculated - Q_setpoint| < 0.001 |Q_setpoint|

Returns:

True if the asset has converged, False otherwise

postprocess() → None

Postprocess after a simulation time step to update internal states.

Returns:

None

pressure_supply: float

The supply pressure of the asset [Pa].

set_pressure_supply(pressure_supply: float) → None

Set the supply pressure of the asset.

Parameters:

pressure_supply (float) – The supply pressure of the asset. The pressure should be supplied in Pascal.

set_setpoints(setpoints: dict) → None

Set the setpoints of the asset.

Parameters:

setpoints (Dict) – The setpoints of the asset in a dictionary, as “property_name”: value pairs.

temperature_in: float

The inlet temperature of the asset [K].

temperature_out: float

The outlet temperature of the asset [K].

thermal_production_required: float | None

The thermal production required by the asset [W].

update() → None

Update the asset properties to the results from the previous (timestep) simulation.

Sets the values of the outlet temperature, inlet temperature and heat demand to the values of the previous simulation. In addition, the mass flow rate is set to the value of the previous simulation.

write_to_output() → None

Method to write time step results to the output dict.

The output list is a list of dictionaries, where each dictionary represents the output of the asset for a specific timestep.
